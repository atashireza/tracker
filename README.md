# 📍 سامانه برداشت میدانی فیبر نوری

ابزار وب برای ثبت موقعیت GPS حین حرکت در مسیر پروژه‌های کبرنوری.

## امکانات
- 📌 ثبت نقاط با دقت GPS
- 🛤️ ترسیم مسیر روی نقشه
- 📐 تبدیل مختصات WGS84 به UTM
- 📥 خروجی CSV و DXF (اتوکد)
